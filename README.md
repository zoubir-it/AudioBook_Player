Audiobook Player 🎧

A sleek and simple audiobook player built with Python, pyglet for audio playback, and tkinter for the graphical interface.

Easily load, play, and navigate through your .wav audiobook files.



✨ Features

🎵 Play, Stop, Next, Previous controls.



📂 Automatic loading of all .wav files from the audio/ folder.



📋 Interactive playlist displayed in a listbox – click a file to play.



🎚️ Clean GUI with visual feedback (current playing status).



🔁 Looping navigation – after the last track, it wraps to the first.



📋 Requirements

Python 3.x



pyglet (for audio playback)



tkinter (usually included with Python)
